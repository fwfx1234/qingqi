启动器

主界面只负责输入、搜索、选择、打开。

主界面常用功能

打开时可见
搜索输入输入插件名、命令前缀、应用名、Quick Launch 动作名
结果列表显示图标、标题、副标题、类型标记
键盘操作上/下选择，Enter 打开，Esc 关闭
鼠标操作点击结果打开；设置入口只作为系统设置快捷入口

结果类型

来自 PluginRuntime commands
插件打开JSON、二维码、剪贴板、系统设置等固定插件入口
动态动作Quick Launch 动作、App Launcher 应用索引等动态命令
前缀匹配api / clip / json / qr / run / app 等，仅影响排序和匹配

软件快速启动

常用只展示应用搜索、列表和打开；索引路径、图标缓存、刷新策略放配置。

主界面常用功能

打开插件时可见
搜索按应用名、bundle id、别名过滤
应用列表图标、名称、路径/Bundle ID、打开按钮
分页/虚拟列表大列表不一次性渲染全部结果
打开应用双击行或点击“打开”

配置中的功能

低频设置
刷新索引手动重扫 /Applications 与用户 Applications
索引路径配置扫描目录和忽略目录
图标缓存缓存、重建、失败回退策略
启动器注册控制是否注册 app: 动态命令

系统设置

所有插件低频功能统一收进设置页或插件自己的配置弹层。

常用设置

直接展示
主题模式浅色、深色、跟随系统；持久化 ThemeStore
插件窗口保留设置 inline session 保留数量
插件导入选择目录、选择 ZIP，校验 manifest 后导入
macOS 权限辅助功能、剪贴板、文件访问等状态检查和跳转

高级维护

配置/诊断区
应用索引维护手动重扫、清理图标缓存
数据路径展示 AppPaths、数据库路径、插件状态
日志诊断展示后台服务状态、最近错误、warning 收敛目标

主界面常用功能

打开插件时可见
搜索动作按名称、描述、关键词、前缀过滤动作

运行配置环境变量、超时、反馈方式
启动器注册关键词、前缀
