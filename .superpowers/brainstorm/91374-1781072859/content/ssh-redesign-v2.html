SSH 远程管理 — 界面重设计

左侧 Profile 卡片 · Session Tab · 文件树+终端 · 整体配色

一、当前问题总结

🔴 需改进

  • Profile 卡片层次感弱，hover 无反馈
  • Session Tab 样式简陋，不像浏览器标签
  • 文件树/终端之间无明显视觉分隔
  • 缺少圆角、阴影等现代 UI 元素
  • 配色平淡，缺少视觉重心
  • 协议徽章不突出

🟢 改进方向

  • 毛玻璃卡片 + 微妙边框 + 圆角
  • 浏览器风格 Tab（圆角顶部 + 底部无边框）
  • 可拖拽分割线分隔文件树/终端
  • 状态指示器更精致（脉动动画）
  • 统一间距系统（4/8/12/16/24px）
  • hover/active 状态更明显

三、设计方向选择 — 哪种风格？

同样的布局，三种不同的视觉风格。选中你喜欢的

A

极简暗色

深灰基调（#0d0d1a → #1a1a2e），蓝灰色边框，低对比度，专注内容

B

毛玻璃质感 (推荐)

透明底 + backdrop-filter blur，半透明卡片叠加层，现代 macOS 风格

C

高对比振动 (VS Code 风格)

深蓝底 + 青色强调色，高饱和度边框，IDE 风格，信息密度高
